AI Immigration
Application Assistant App

AI-powered Canadian Visa Application Simplification. An web application streamlines the process by automatically filling required forms and generating a comprehensive list of necessary documents, enhancing accessibility and efficiency.

Project Overview

This project aimed to simplify the Canadian visa application process using AI. The web application auto-fills required forms and provides a list of necessary documents, making the application process more accessible. An admin portal also allows administrators to assist users in completing their applications.

My Role

As the lead web designer, I worked closely with software engineers to understand the technical aspects of data handling in the backend. I designed user-friendly, responsive webpages, ensuring seamless integration of design and functionality.
